# Build boilerplate

A webpack build boilerplate that includes:

- Dev and production configs
- Dev server with HMR
- JavaScript transpilation
- React
- SCSS compilation
- Linting

## Setup

1. Clone the repo
1. `npm i`

You're good to go!

## Usage

1. `npm run dev` for dev server with HMR
1. `npm run build` for production build
